# AvatarFallback

## Overview
AvatarFallback renders a textual placeholder when an avatar image is missing.
It assigns a deterministic background color based on a string (for example, a user’s e-mail).

## Styling Guidelines
All examples use Tailwind CSS.
No Contember-specific theme classes are required for AvatarFallback.

## Core Concepts
- You supply a child node, usually the user’s initial.
- The `avatarFallbackColorString` prop hashes the string into a background color.
- The component accepts standard `className` for additional Tailwind styles.

## Quick Start
```tsx
import { Avatar, AvatarFallback } from '~/lib/ui/avatar'

export const UserAvatar = ({ user }) => (
	<Avatar>
		<AvatarFallback
			avatarFallbackColorString={user.email}
			className="w-10 h-10 rounded-lg text-white flex items-center justify-center"
		>
			{user.name[0]}
		</AvatarFallback>
	</Avatar>
)
```

## Props Reference
| Name | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| avatarFallbackColorString | string | No |  | Source string used to generate the background color. |
| className | string | No |  | Additional Tailwind classes. |
| children | ReactNode | Yes |  | Text displayed inside the fallback. |

## Examples

### Display initial in a list
```tsx
{users.map(user => (
	<AvatarFallback
		key={user.id}
		avatarFallbackColorString={user.email}
		className="w-8 h-8 rounded-full text-xs font-medium text-white flex items-center justify-center"
	>
		{user.name[0]}
	</AvatarFallback>
))}
```
The map renders a small circular fallback for each user, using the e-mail to ensure consistent colors.

### Theming with dark mode
```tsx
<AvatarFallback
	avatarFallbackColorString={user.email}
	className="w-12 h-12 rounded-lg text-white dark:text-gray-800"
>
	{user.name[0]}
</AvatarFallback>
```
Tailwind’s `dark:` modifier changes the text color when dark mode is active.

## Best Practices & Gotchas
- Always pass a stable string (e-mail, UUID) to keep the color consistent across sessions.
- Keep children to one or two characters; longer strings will not fit.
- Combine with `Avatar` for graceful fallback when the image fails to load.
