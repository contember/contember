# BlockEditorInner

## Styling Guidelines
* All examples use Tailwind CSS utility classes.
* No Contember-specific theme classes are required.

## Core Concepts
* Acts as the inner surface of a `BlockEditor`.
* Expects to live inside both `BlockEditor` and `SortableBlockEditor`.
* Shows an optional placeholder when empty.

## Quick Start
```tsx
import { BlockEditor, SortableBlockEditor, BlockEditorInner } from '@contember/react-editors'

export const PageEditor = () => (
	<BlockEditor plugins={[]}>
		<SortableBlockEditor>
			<BlockEditorInner placeholder="Start writing here…" className="prose max-w-none" />
		</SortableBlockEditor>
	</BlockEditor>
)
```

## Best Practices & Gotchas
* Always nest `BlockEditorInner` inside `BlockEditor` and `SortableBlockEditor`.
* Pass a short, helpful `placeholder` to improve user onboarding.
* Combine with Tailwind’s typography utilities (`prose`) for readable output.

## Further Reading
* `BlockEditor` – top-level rich text context.
* `SortableBlockEditor` – drag-and-drop ordering of blocks.
