# CardFooter

## Overview
CardFooter represents the bottom region of a Contember `Card`.
You typically place form actions, navigation links, or secondary information inside it.

## Styling Guidelines
* All snippets use Tailwind CSS.
* No dedicated Contember theme classes are required; apply any Tailwind utility such as `ml-auto` or `border-t` directly on the component.

## Core Concepts
* CardFooter is only meaningful inside a `Card` layout.
* It renders its children in a flex container, so Tailwind spacing utilities work predictably.
* Combine `CardHeader`, `CardContent`, and `CardFooter` for a complete card.

## Quick Start
```tsx
import { Card, CardHeader, CardContent, CardFooter, CardTitle } from '~/lib/ui/card'

export const SimpleCard = () => (
  <Card className="w-96">
    <CardHeader>
      <CardTitle className="text-xl">Card title</CardTitle>
    </CardHeader>

    <CardContent>
      <p className="text-gray-700">Main content goes here.</p>
    </CardContent>

    <CardFooter className="flex gap-2">
      <button className="btn btn-primary">Save</button>
      <button className="btn btn-secondary ml-auto">Cancel</button>
    </CardFooter>
  </Card>
)
```

## Examples

### Footer with “Back to Login” link
```tsx
<CardFooter>
  <Link to="index">
    <AnchorButton variant="link" className="ml-auto">
      {dict.tenant.login.backToLogin}
    </AnchorButton>
  </Link>
</CardFooter>
```
This pattern centers the main content earlier in the card and aligns the navigation link to the far right using `ml-auto`.

### Form actions
```tsx
<CardFooter className="flex justify-end gap-3">
  <button className="btn btn-outline">Reset</button>
  <button className="btn btn-primary">Submit</button>
</CardFooter>
```
Grouping buttons inside `CardFooter` separates form logic from the body, improving visual hierarchy.

## Best Practices & Gotchas
* Keep primary action buttons on the right (`ml-auto`) to follow common UX conventions.
* Add `border-t` to the footer to visually separate it from content.
* Avoid placing large blocks of text; keep it concise.
