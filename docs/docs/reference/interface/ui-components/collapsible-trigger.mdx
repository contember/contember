## Overview
CollapsibleTrigger renders an interactive element that toggles the open state of a Contember `Collapsible`.
Use it when you need users to show or hide content on demand.

## Styling Guidelines
* All examples use Tailwind CSS.
* No special Contember theme classes are required, but feel free to add your own utilities.

## Core Concepts
* CollapsibleTrigger controls visibility of a sibling `CollapsibleContent`.
* It should live inside a `Collapsible` provider.
* You can wrap any element in `CollapsibleTrigger` by using the `asChild` prop from Radix (supported internally).

## Quick Start
```tsx
import { Collapsible, CollapsibleContent, CollapsibleTrigger } from '~/lib/ui/collapsible'

export const BasicCollapsible = () => (
	<Collapsible className="w-full max-w-sm rounded bg-gray-50 p-4 shadow">
		<CollapsibleTrigger className="flex w-full items-center justify-between rounded bg-white px-4 py-2 text-sm font-medium">
			<span>Details</span>
			<span className="text-gray-500">&gt;</span>
		</CollapsibleTrigger>

		<CollapsibleContent className="mt-2 text-sm text-gray-700">
			Here are the hidden details. Click the header again to hide.
		</CollapsibleContent>
	</Collapsible>
)
```

## Examples

### Sidebar Menu
```tsx
const SidebarSection = ({ icon, label, children }: { icon: React.ReactNode; label: string; children: React.ReactNode }) => (
	<Collapsible className="group/collapsible">
		<CollapsibleTrigger asChild>
			<button className="flex w-full items-center gap-2 rounded px-3 py-2 text-left hover:bg-gray-100">
				{icon}
				<span>{label}</span>
				<span className="ml-auto transition-transform group-open/collapsible:rotate-90">
					<ChevronRight className="h-4 w-4 text-gray-600" />
				</span>
			</button>
		</CollapsibleTrigger>

		<CollapsibleContent className="pl-6">
			{children}
		</CollapsibleContent>
	</Collapsible>
)
```
The trigger wraps a custom button. The `group-open/collapsible:rotate-90` utility animates the chevron based on the open state.

### Controlled Open State
```tsx
const [open, setOpen] = useState(false)

<Collapsible open={open} onOpenChange={setOpen}>
	<CollapsibleTrigger className="rounded bg-blue-600 px-4 py-2 text-white">
		Toggle Content
	</CollapsibleTrigger>
	<CollapsibleContent className="mt-2 border-l-2 border-blue-600 pl-4">
		This content is controlled from outside.
	</CollapsibleContent>
</Collapsible>
```
Manage the `open` state from your component when you need external control.

## Best Practices & Gotchas
* Always place `CollapsibleTrigger` inside a `Collapsible`; it will not work standalone.
* Use `asChild` to avoid extra DOM nodes when you need semantic markup.
* Add transition classes to `CollapsibleContent` for smooth animations.
* Keep the trigger size large enough for accessible click targets.
