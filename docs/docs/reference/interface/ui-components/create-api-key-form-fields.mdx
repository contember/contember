# CreateApiKeyFormFields

## Overview
`CreateApiKeyFormFields` renders all input elements required for the `CreateApiKeyForm` component.
You use it inside a `<form>` element to capture API-key metadata tied to a specific Contember project.

## Styling Guidelines
* All examples use Tailwind utility classes.
* The component itself is unstyled; wrap it in your own elements or Contember UI primitives to match your theme.

## Core Concepts
* Composable form fields – drop the component anywhere inside a `<CreateApiKeyForm>`.
* Prop-driven context – you must pass the current `projectSlug`.
* Uncontrolled styling – you control layout and spacing with Tailwind.

## Quick Start
```tsx
import { CreateApiKeyForm, CreateApiKeyFormFields } from '~/lib/tenant'

export const Page = () => {
  const projectSlug = 'my-project'

  return (
    <CreateApiKeyForm projectSlug={projectSlug}>
      <form className="grid gap-4">
        <CreateApiKeyFormFields projectSlug={projectSlug} />
      </form>
    </CreateApiKeyForm>
  )
}
```

| Name | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| projectSlug | `string` | Yes | – | The slug of the project for which you are creating the API key. |

## Examples

### Full page with list refresh
```tsx
() => {
  const projectSlug = useProjectSlug()!
  const showToast = useShowToast()
  const memberListController = useRef<MemberListController>()

  return (
    <>
      <Slots.Title>
        <Title icon={<KeyRoundIcon />}>API keys</Title>
      </Slots.Title>

      <div className="flex flex-col gap-4">
        <Card className="w-[40rem] max-w-full">
          <CardHeader>
            <CardTitle className="text-2xl">Create API key</CardTitle>
          </CardHeader>
          <CardContent>
            <CreateApiKeyForm
              projectSlug={projectSlug}
              initialMemberships={[{ role: 'admin', variables: [] }]}
              onSuccess={args => {
                showToast(
                  <ToastContent title="API key created">
                    <Input value={args.result.apiKey.token} type="text" />
                  </ToastContent>,
                  { type: 'success' },
                )
                memberListController.current?.refresh()
              }}
            >
              <form className="grid gap-4">
                <CreateApiKeyFormFields projectSlug={projectSlug} />
              </form>
            </CreateApiKeyForm>
          </CardContent>
        </Card>

        <Card className="w-[40rem] max-w-full">
          <CardHeader>
            <CardTitle className="text-2xl">API keys</CardTitle>
          </CardHeader>
          <CardContent>
            <ApiKeyList controller={memberListController} />
          </CardContent>
        </Card>
      </div>
    </>
  )
}
```
The example shows how to:
1. Provide `projectSlug`.
2. Display feedback in a toast.
3. Refresh a related list once the form succeeds.

## Best Practices & Gotchas
* Always pass the same `projectSlug` to both `CreateApiKeyForm` and `CreateApiKeyFormFields`.
* Wrap fields in a `<form>` with `grid gap-*` classes for consistent spacing.
* Keep token display secure—avoid logging raw tokens.
* Refresh dependent lists after key creation to show up-to-date data.
