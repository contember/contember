# DataGridEnumListCell

## Overview
`DataGridEnumListCell` renders a data-grid cell that shows one or more enum values.
The cell opens a tooltip that lets you trigger filter actions linked to the displayed enum.

## Styling Guidelines
* All examples use Tailwind CSS utility classes.
* The component applies Contember’s default DataGrid theme classes automatically; you do not need to add any custom theme class.

## Core Concepts
* Enum values are rendered as readable labels.
* The tooltip provides quick filter actions tied to the enum values.
* You can override enum labels via the `options` prop.
* Filtering logic relies on Contember DataGrid filters (`filterName`).

## Quick Start
```tsx
import { DataGrid, DataGridEnumListCell } from '@contember/react-datagrid'

export const OrdersGrid = () => (
	<DataGrid className="w-full">
		<DataGridEnumListCell
			field="status"
			className="px-4 py-2"
		/>
	</DataGrid>
)
```

## Detailed Guide
```tsx
import { DataGrid, DataGridEnumListCell } from '@contember/react-datagrid'

const statusLabels = {
	pending: 'Pending',
	shipped: 'Shipped',
	delivered: 'Delivered',
	cancelled: 'Cancelled',
}

export const OrdersGrid = () => (
	<DataGrid className="w-full">
		<DataGridEnumListCell
			field={{ field: 'status' }}
			filterName="statusFilter"
			options={statusLabels}
			tooltipActions={
				<button
					className="mt-2 w-full rounded bg-red-600 px-2 py-1 text-white hover:bg-red-700"
					onClick={() => alert('Custom action')}
				>
					Mark as cancelled
				</button>
			}
			className="px-4 py-2"
		/>
	</DataGrid>
)
```

## Props Reference
| Name | Type | Required | Default | Description |
| ---- | ---- | -------- | ------- | ----------- |
| `field` | `string` &#124; `{ field: string; hasOneRelationPath?: SugarableHasOneRelation &#124; SugarableHasOneRelation[] }` | Yes | — | Field to be displayed. |
| `filterName` | `string` | No | Derived from `field` | Filter identifier. |
| `options` | `Record<string, React.ReactNode>` | No | Enum metadata | Enum labels shown in the tooltip. |
| `tooltipActions` | `React.ReactNode` | No | — | Custom actions rendered inside the tooltip. |

## Examples

### Custom Enum Labels
```tsx
<DataGridEnumListCell
	field="priority"
	options={{ low: 'Low', medium: 'Medium', high: 'High' }}
	className="px-4 py-2"
/>
```
Use `options` to map enum values to user-friendly labels.

### Tooltip with Extra Action
```tsx
<DataGridEnumListCell
	field="status"
	tooltipActions={<ExportButton />}
	className="px-4 py-2"
/>
```
Embed any React node into the tooltip for richer interactions such as exporting or bulk updates.

## Best Practices & Gotchas
* Provide `options` when enum values are cryptic; keep UI clear.
* Reuse `filterName` across multiple cells to share the same filter state.
* Keep custom tooltip actions concise; overcrowded tooltips harm usability.
* Do not mutate enum lists client-side; treat them as read-only display data.
