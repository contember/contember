# DataGridIsDefinedFilterControls

## Overview
DataGridIsDefinedFilterControls is an internal Contember component used inside a DataGrid. It renders controls that let users filter records based on whether a field is defined.

## Styling Guidelines
All examples use Tailwind. No additional Contember theme classes are required.

## Core Concepts
- Internal component, not intended for direct public use.
- Works exclusively with Contember’s DataGrid.
- Filters records by “defined” vs. “undefined” field state.

## Quick Start
```tsx
import { DataGridIsDefinedFilterControls } from '@contember/react-datagrid'

export const Example = () => (
	<div className="p-4">
		<DataGridIsDefinedFilterControls />
	</div>
)
```
