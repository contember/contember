# DataGridRelationFilterControls

## Overview
`DataGridRelationFilterControls` renders action buttons for relation filters inside Contember DataGrid.
You embed it inside `DataViewHasManyFilter` next to `DataGridRelationFilteredItemsList` to let users add or remove related entities in a filter.

## Styling Guidelines
* All examples below use Tailwind CSS.
* No extra Contember theme classes are required—Tailwind utilities are enough.

## Core Concepts
* Works only inside a relation filter (`DataViewHasManyFilter`).
* Expects preview components passed as `children`.
* Displays buttons like “Add” and “Clear” for the current relation filter.

## Quick Start
```tsx
import {
	DataGridColumn,
	DataGridRelationFilterControls,
	DataGridRelationFilteredItemsList,
} from '~/lib/datagrid'

<DataViewHasManyFilter field="tags">
	<div className="border rounded p-2 max-w-60 flex flex-col gap-2">
		<div className="flex flex-wrap gap-2">
			<DataGridRelationFilteredItemsList>
				<TagPreview />
			</DataGridRelationFilteredItemsList>
		</div>
		<DataGridRelationFilterControls>
			<TagPreview />
		</DataGridRelationFilterControls>
	</div>
</DataViewHasManyFilter>
```

## Examples

### Filter companies by locality
```tsx
<DataViewHasManyFilter field="locality">
	<div className="border rounded p-2 max-w-60 flex flex-col gap-2">
		<div className="flex flex-wrap gap-2">
			<DataGridRelationFilteredItemsList>
				<CompanyLocalityPreview />
			</DataGridRelationFilteredItemsList>
		</div>

		<DataGridRelationFilterControls>
			<CompanyLocalityPreview />
		</DataGridRelationFilterControls>
	</div>
</DataViewHasManyFilter>
```
In this snippet you:
* Show selected localities inside the filter area.
* Offer controls for adding or clearing localities.

### Filter medical providers inside column filter
```tsx
<DataViewHasManyFilter field="medicalProviders" name="medicalProviders">
	<div className="border rounded p-2 max-w-60 flex flex-col gap-2">
		<div className="flex flex-wrap gap-2">
			<DataGridRelationFilteredItemsList>
				<MedicalProviderLocalityPreview />
			</DataGridRelationFilteredItemsList>
		</div>

		<DataGridRelationFilterControls>
			<MedicalProviderLocalityPreview />
		</DataGridRelationFilterControls>
	</div>
</DataViewHasManyFilter>
```
This pattern lets users pick multiple medical providers while viewing the grid.

## Best Practices & Gotchas
* Always wrap the component in `DataViewHasManyFilter`.
* Pass the same preview component as `children` to both `DataGridRelationFilteredItemsList` and `DataGridRelationFilterControls`.
* Keep the filter panel width narrow (`max-w-60`) to avoid layout issues.
