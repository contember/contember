## Styling Guidelines
All code examples in this documentation use Tailwind CSS classes.
There are no additional Contember theme classes required for `DataGridShowFiltersContext`.

## Core Concepts
- React Context for sharing “show filters” state across DataGrid components.
- Composition with existing Contember DataGrid elements.

## Best Practices & Gotchas
- Wrap related DataGrid components with the same `DataGridShowFiltersContext.Provider` to avoid state mismatches.
- Keep filter-state logic close to the DataGrid to simplify debugging.
- Unmount the provider when the DataGrid leaves the page to reset filter visibility.
