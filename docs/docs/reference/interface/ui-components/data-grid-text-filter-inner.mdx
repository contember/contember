## Overview
`DataGridTextFilterInner` is an internal utility that powers text-based filtering inside Contember data grids.
You normally interact with higher-level filter components, but the inner component can be useful for custom integrations.

## Styling Guidelines
All examples use Tailwind CSS utility classes.
No additional Contember theme classes are required for this component.

## Core Concepts
- Internal component: not meant for direct public use.
- Provides a text input bound to the data-grid filtering logic.
- Accepts **no props**; all configuration happens in parent components.

## Quick Start
```tsx
import { DataGridTextFilterInner } from '@contember/react-datagrid'

export const BasicFilter = () => (
  <div className="p-4 bg-gray-50">
    <DataGridTextFilterInner />
  </div>
)
```
