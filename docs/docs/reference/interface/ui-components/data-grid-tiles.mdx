## Overview
`DataGridTiles` renders records in a responsive grid inside `DataView`.
Use it when you want a card-style layout instead of rows or tables.

The component adds only layout.
You supply the tile markup through the `children` prop.

## Styling Guidelines
* All examples use Tailwind CSS.
* `DataGridTiles` itself needs no special Contember theme classes.

## Core Concepts
* `DataGridTiles` must live inside `DataGridLoader`.
* The grid adapts via CSS grid. Change columns with the `className` prop.
* Every direct child becomes one tile.
* Combine with `DataGridTable` for view switching.

## Quick Start
```tsx
import { DataGrid, DataGridLoader, DataGridTiles } from '~/lib/datagrid'

<DataGrid entities="Article">
	<DataGridLoader>
		<DataGridTiles className="grid-cols-1 sm:grid-cols-2 gap-4">
			<UseEntity render={article => (
				<div className="p-4 rounded shadow bg-white">
					<h2 className="text-lg font-semibold">{article.getField('title')}</h2>
					<p className="text-sm text-gray-500">{article.getField('publishedAt')}</p>
				</div>
			)} />
		</DataGridTiles>
	</DataGridLoader>
</DataGrid>
```

## Detailed Guide
### Custom column count
Adjust columns with Tailwind utilities.
```tsx
<DataGridTiles className="md:grid-cols-[repeat(auto-fill,minmax(10rem,1fr))] gap-6">
	{/* tiles here */}
</DataGridTiles>
```

### Combining tiles and table views
```tsx
<DataGridLoader>
	<DataGridTiles>
		<CustomTile />
	</DataGridTiles>

	<DataGridTable>
		<CustomColumn />
	</DataGridTable>
</DataGridLoader>
```

## Props Reference
| Name | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| children | React.ReactNode | Yes | — | Elements rendered as individual tiles. |
| className | string | No | — | Extra Tailwind or custom classes for the grid. |
| *Inherited* | — | — | — | All props of HTML `div`. |

## Examples
### Image picker grid
```tsx
<DataGridTiles className="md:grid-cols-[repeat(auto-fill,minmax(10rem,1fr))]">
	<UseEntity render={it => (
		<div
			className="relative border rounded shadow hover:shadow-md"
			onClick={() => connect(it)}
		>
			<UploadedImageView />
		</div>
	)} />
</DataGridTiles>
```
Selecting a tile calls `connect` to link the chosen entity.

### Mixed layouts
```tsx
<DataGridLoader>
	<DataGridTiles>
		<CompactTile />
	</DataGridTiles>
	<DataGridTable>
		<VerboseColumn />
	</DataGridTable>
</DataGridLoader>
```
Users switch between visual and tabular views without reloading data.

## Best Practices & Gotchas
* Always wrap `DataGridTiles` in `DataGridLoader`; otherwise no data loads.
* Keep tiles lightweight; large components hurt scroll performance.
* Control gaps and columns with `className` instead of inline styles.
* Combine with `DataViewLayout` for user selectable layouts.
* Supply a stable key on tile roots when mapping collections.
