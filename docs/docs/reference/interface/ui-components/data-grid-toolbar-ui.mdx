## Overview
`DataGridToolbarUI` wraps filtering, sorting, and other control components inside a `DataGrid`.
It gives you a predictable flex-box layout so you can compose any combination of query filters, dropdowns, and buttons.

## Styling Guidelines
* All examples use Tailwind CSS.
* `DataGridToolbarUI` itself adds no theme classes, so you control spacing with Tailwind utilities.

## Core Concepts
* Place `DataGridToolbarUI` as the first direct child of `DataGrid`.
* Render filter components (`DataGridQueryFilter`, `DataGridHasManyFilter`, …) inside it.
* Layout is flex-box; add your own wrappers when you need extra alignment.

## Quick Start
```tsx
import { DataGrid, DataGridToolbarUI, DataGridQueryFilter } from '~/lib/datagrid'

export const CoursesTable = () => (
	<DataGrid entities="Course">
		<DataGridToolbarUI>
			<div className="flex flex-1 flex-wrap gap-2">
				<DataGridQueryFilter />
			</div>
		</DataGridToolbarUI>

		{/* ...your DataGridLoader and row renderers... */}
	</DataGrid>
)
```

## Examples

### Filtering marketplace courses
The toolbar groups a free-text filter with a category filter.

```tsx
<DataGridToolbarUI>
	<div className="flex flex-1 flex-wrap gap-2">
		<DataGridQueryFilter />
		<DataGridHasManyFilter
			field="categories"
			options="CourseCategory[courseModules.publishOnMarketPlaceAt!=null]"
			label="Category"
		>
			<CourseCategoryPreview />
		</DataGridHasManyFilter>
	</div>
</DataGridToolbarUI>
```

The `flex flex-1 flex-wrap gap-2` classes make the controls wrap nicely on small screens.

### Adding a sorting dropdown
```tsx
<DataGridToolbarUI>
	<div className="flex flex-1 flex-wrap gap-2">
		<DataGridQueryFilter />
		<DataGridSortingDropdown
			options={[
				{ label: 'Newest', field: 'createdAt', direction: 'desc' },
				{ label: 'Oldest', field: 'createdAt', direction: 'asc' },
			]}
		/>
	</div>
</DataGridToolbarUI>
```

## Best Practices & Gotchas
* Always wrap inner controls in a flex container; the component does not add spacing itself.
* Keep the toolbar as the first child of `DataGrid` so keyboard focus order is predictable.
* Use `flex-1` to let controls expand and align with the grid’s width.
* Combine small filters into one line to avoid vertical space waste on mobile.
