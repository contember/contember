# DropdownMenuLabel

## Overview
`DropdownMenuLabel` renders a styled label or header inside a dropdown menu.
Use it to visually group related menu items or present contextual information.

## Styling Guidelines
* All examples use Tailwind CSS classes.
* No additional Contember theme classes are required for this component.

## Core Concepts
* `DropdownMenuLabel` is a presentational component.
* It typically wraps headings, avatars, or small layout grids.
* It lives inside a `DropdownMenuContent` element.
* You can pass regular children to control its content.

## Quick Start
```tsx
import {
  DropdownMenu,
  DropdownMenuTrigger,
  DropdownMenuContent,
  DropdownMenuLabel,
  DropdownMenuItem,
} from '~/lib/ui/dropdown'

export default function Example() {
  return (
    <DropdownMenu>
      <DropdownMenuTrigger className="px-3 py-2 bg-primary text-white rounded">
        Open menu
      </DropdownMenuTrigger>

      <DropdownMenuContent className="w-56">
        <DropdownMenuLabel className="text-xs text-gray-500 uppercase tracking-wide px-3 py-1">
          Navigation
        </DropdownMenuLabel>

        <DropdownMenuItem className="px-3 py-2 hover:bg-gray-100">
          Dashboard
        </DropdownMenuItem>
      </DropdownMenuContent>
    </DropdownMenu>
  )
}
```

## Examples

### User Profile Header
Show a user avatar and email above account actions.

```tsx
<DropdownMenuLabel className="p-0 font-normal">
  <div className="flex items-center gap-2 pl-3 pr-1 py-1.5 text-left text-sm">
    <Avatar className="h-8 w-8 rounded-lg">
      <AvatarFallback avatarFallbackColorString={userEmail} className="rounded-lg">
        {userInitial}
      </AvatarFallback>
    </Avatar>

    <div className="grid flex-1 text-left text-sm leading-tight">
      <span className="truncate font-semibold">{userEmail}</span>
    </div>
  </div>
</DropdownMenuLabel>
```

Explanation
* The label occupies the full width of the menu.
* Utility classes provide spacing and typography.
* Child layout is fully controlled by you.

## Best Practices & Gotchas
* Keep label content concise—avoid long sentences.
* Use semantic HTML elements inside the label when possible.
* Add padding directly on the label to align with menu items.
* Do not place interactive elements (buttons, links) inside a label.

## Further Reading
* `DropdownMenuContent` – container for menu items.
* `DropdownMenuItem` – clickable action inside the menu.
* `Avatar` – component used in the profile header example.
