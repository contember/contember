# DropdownMenuPortal

## Overview
DropdownMenuPortal renders its children into a React Portal.
Use it inside a dropdown menu to ensure that the menu content escapes the stacking context of parent elements.

## Styling Guidelines
* All examples use Tailwind CSS utility classes.
* No special Contember theme classes are required.

## Core Concepts
* Portals let you render elements outside the normal DOM hierarchy.
* Placing dropdown content in a portal avoids `overflow:hidden`, z-index, and stacking issues.
* The component itself forwards all props to the underlying React Portal.

## Quick Start
```tsx
import { DropdownMenuPortal } from '@contember/react-components'

export const SimpleDropdown = () => (
	<DropdownMenuPortal>
		<div className="bg-white shadow-lg rounded p-2">
			Menu content
		</div>
	</DropdownMenuPortal>
)
```

## Best Practices & Gotchas
* Always wrap dropdown content in `DropdownMenuPortal` to avoid clipping.
* Add Tailwind classes like `z-50` when you need to raise the stacking context.
* Keep portal content lightweight to minimize DOM updates.
