## Overview
DropdownMenuRadioGroup wraps related radio items inside a dropdown menu. It ensures that only one item in the group can be active at any time.

Use it when you need a compact, mutually-exclusive choice list in a menu.

## Styling Guidelines
* All examples below rely on Tailwind CSS utilities.
* The component requires no specific Contember theme classes out of the box.

## Core Concepts
* A radio group limits selection to a single child item.
* Works only with radio-type menu items.
* Designed for composition inside a parent `DropdownMenu`.

## Quick Start
```tsx
import { DropdownMenuRadioGroup } from '@contember/react-components'

export const LanguageSwitcher = () => (
  <DropdownMenuRadioGroup>
    {/*
      Insert <DropdownMenuRadioItem> components here, for example:
      <DropdownMenuRadioItem value="en">English</DropdownMenuRadioItem>
    */}
  </DropdownMenuRadioGroup>
)
```

## Best Practices & Gotchas
* Keep only radio-style items as direct children.
* Avoid mixing actions or checkboxes in the same group.
