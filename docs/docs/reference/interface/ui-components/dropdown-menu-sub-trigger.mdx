## Styling Guidelines
All examples use Tailwind for utility-first styling.  
This component requires no special Contember theme classes.

## Core Concepts
- Acts as a trigger that opens a nested dropdown menu.  
- Must be placed inside `DropdownMenuSub`.  
- Receives focus and hover states from Tailwind utility classes.

## Quick Start
```tsx
import {
  DropdownMenu,
  DropdownMenuSub,
  DropdownMenuSubTrigger,
  DropdownMenuContent,
} from '@contember/ui'

export const BasicSubTrigger = () => (
  <DropdownMenu>
    <DropdownMenuSub>
      <DropdownMenuSubTrigger className="flex items-center gap-2 px-2 py-1 hover:bg-gray-100">
        More options
      </DropdownMenuSubTrigger>

      <DropdownMenuContent className="bg-white shadow-lg rounded">
        {/* Sub-menu items go here */}
      </DropdownMenuContent>
    </DropdownMenuSub>
  </DropdownMenu>
)
```

## Best Practices & Gotchas
- Place `DropdownMenuSubTrigger` directly inside `DropdownMenuSub`.  
- Use clear, concise text so users know a submenu exists.  
- Add focus and hover Tailwind classes for keyboard accessibility.  
- Keep submenu content concise to avoid deep nesting.
