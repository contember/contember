# FormLabelWrapperUI

## Overview
FormLabelWrapperUI wraps one or more `FormLabelUI` elements to keep them aligned and evenly spaced.
Internally it renders a flex container, so labels line up neatly in complex forms.

## Styling Guidelines
* All snippets use Tailwind CSS.
* The component already applies the base `flex` class; you rarely need extra layout utilities.

## Core Concepts
* The wrapper is purely presentational—no state or logic.
* Children render exactly as passed; they are not modified.
* Use it inside `FormContainerUI` for consistent padding and margins.
* Combine it with other Contember form primitives like `FormDescriptionUI`.

## Quick Start
```tsx
import { FormLabelWrapperUI, FormLabelUI } from '~/lib/form'

export const EmailFieldLabel = () => (
  <FormLabelWrapperUI>
    <FormLabelUI required>Email address</FormLabelUI>
  </FormLabelWrapperUI>
)
```

## Examples

### Toggleable date selector
This pattern shows how the wrapper keeps a custom label and a `Switch` aligned.
```tsx
<FormLabelWrapperUI>
  <FormLabelUI className="flex items-start">
    <Switch checked={isActive} onCheckedChange={setIsActive} />
    <span className="ml-2">Activate from date</span>
  </FormLabelUI>
</FormLabelWrapperUI>
```
The `flex` layout ensures the switch and text stay on the same baseline.

### Entity preview inside a relation
```tsx
<FormLabelWrapperUI>
  <FormLabelUI>
    <span>Lecturer</span>
  </FormLabelUI>
</FormLabelWrapperUI>

<HasOne field="lecturer">
  <LecturerPreview />
</HasOne>
```
The wrapper isolates the label, so relation previews below it do not affect alignment.

### Multi-locale selector header
```tsx
<FormLabelWrapperUI>
  {label && <FormLabelUI>{label}</FormLabelUI>}
</FormLabelWrapperUI>

<DimensionsSwitcher
  options="Locale"
  slugField="code"
  dimension="locale"
  orderBy={{ code: 'asc' }}
  isMulti
>
  <LocalePreview />
</DimensionsSwitcher>
```
Here the wrapper keeps the optional `label` element centered even when the locale picker renders dynamic content beneath it.

## Best Practices & Gotchas
* Place only label elements inside; avoid form controls.
* Add margin or padding on the wrapper, not individual labels, to keep spacing consistent.
* Do not nest multiple wrappers unnecessarily—it increases DOM depth without benefit.
* Remember that `FormLabelUI` can accept `required` to show an asterisk; the wrapper does not forward props.
