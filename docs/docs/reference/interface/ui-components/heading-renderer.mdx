## Quick Start
```tsx
import { HeadingRenderer } from './elements/HeadingRenderer'

// `element` must be an instance of `HeadingElement` obtained from your Slate editor’s node tree.

  <HeadingRenderer element={element} />
</div>
```

## Props Reference
| Name    | Type | Required | Default | Description |
|---------|------|----------|---------|-------------|
