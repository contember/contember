# HorizontalRuleRenderer

## Overview
`HorizontalRuleRenderer` visualizes a horizontal rule (`<hr />`) element within the Contember Slate editor. It converts the internal `HorizontalRuleElement` node into a styled divider that separates content blocks.

The component is presentation-only. It receives the Slate element object and renders a semantic `<hr>` with Tailwind utility classes.

## Styling Guidelines
* All examples use Tailwind CSS.
* No mandatory Contember theme classes exist for this component, but you can layer your own design-system styles on top.

## Core Concepts
* Use the renderer inside a Contember Slate editor context.
* Pass the exact Slate element object (`HorizontalRuleElement`) via the `element` prop.
* Apply Tailwind classes directly to the rendered `<hr>` for styling.

## Quick Start
```tsx
import { HorizontalRuleRenderer } from './elements/HorizontalRuleRenderer'

export const MyEditor = () => (
	<HorizontalRuleRenderer
		element={{ type: 'horizontalRule', children: [] }}
	/>
)
```

## Best Practices & Gotchas
* Always supply the real Slate `HorizontalRuleElement`; passing an empty object causes runtime errors.
* Keep rules visually subtle to avoid overwhelming the document.
* Combine Tailwind classes instead of inline styles for easier theming.
* Wrap the renderer in `React.memo` if you render many rules for performance gains.
