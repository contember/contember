## Overview
`LayoutComponent` arranges your application shell and exposes named slots for common UI areas such as logo, navigation, sidebar, and footer.  
Use it to keep page structure consistent and to swap parts without rewriting markup.

## Styling Guidelines
* All examples use Tailwind CSS classes.  
* `LayoutComponent` itself does not enforce styles, but slot content should respect your Contember theme utilities (e.g. `.content-link`).

## Core Concepts
* Slot-based composition lets you inject UI pieces declaratively.
* You wrap page content in `<LayoutComponent>` once; children render inside the main area.
* Combine with `IdentityLoader` for authenticated layouts.
* Slots are React components, not strings—import them from the same module.
* The component handles responsive layout; you style only the slot contents.

## Quick Start
```tsx
import { LayoutComponent, Slots } from '~/lib/layout/layout'
import { Link } from '@contember/react-router'

export const Page = () => (
	<LayoutComponent>
		<Slots.Logo>
			<Link to="index" className="block p-4 font-bold">Brand</Link>
		</Slots.Logo>

		<Slots.Navigation>
			<nav className="flex gap-4 p-4">
				<Link to="dashboard">Dashboard</Link>
				<Link to="settings">Settings</Link>
			</nav>
		</Slots.Navigation>

		{/* Page content */}
		<main className="p-8">Hello world</main>
	</LayoutComponent>
)
```

### Slots.Logo

### Slots.Footer

## Examples
### Full Application Shell
```tsx
import { LayoutComponent, Slots } from '~/lib/layout/layout'

export const Shell = ({ children }: { children: React.ReactNode }) => (
	<LayoutComponent>
		<Slots.Logo>
			<img src="/logo.svg" alt="Logo" className="h-10 w-auto mx-auto" />
		</Slots.Logo>

		<Slots.Navigation>
			<ul className="flex gap-6 p-4">
				<li><Link to="index">Home</Link></li>
				<li><Link to="articles">Articles</Link></li>
			</ul>
		</Slots.Navigation>

		<Slots.Sidebar>
			<div className="h-full flex flex-col">
				<header className="p-4 border-b">Sidebar header</header>
				<div className="flex-1 p-4">Sidebar content</div>
				<footer className="p-4 border-t">Sidebar footer</footer>
			</div>
		</Slots.Sidebar>

		<Slots.Footer>
			<footer className="text-center p-4 text-sm text-gray-500">
				© 2024 Your Company
			</footer>
		</Slots.Footer>

		{children}
	</LayoutComponent>
)
```
The shell adds a responsive sidebar and a footer. You can override any slot per route if needed.

## Best Practices & Gotchas
* Keep slot content small; avoid large trees inside a slot.
* Do not nest another `LayoutComponent` inside itself.
* Style slots, not the container. The layout already controls spacing.
* Always wrap secure pages in `IdentityLoader` before `LayoutComponent`.
* Remember to import slots from the same file as `LayoutComponent`.
