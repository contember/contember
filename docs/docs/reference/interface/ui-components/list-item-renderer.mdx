## Core Concepts  
* `ListItemRenderer` renders a Slate `ListItemElement`.  
* You must pass the `element` prop obtained from your Slate editor.  

## Quick Start  
```tsx
import { ListItemRenderer } from './elements/ListItemRenderer'

export const Example = ({ element }: { element: ListItemElement }) => (
  <div className="flex">
    <ListItemRenderer element={element} />
  </div>
)
```
