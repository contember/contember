## Styling Guidelines
All examples use Tailwind CSS utility classes. The component does not rely on any Contember-specific theme classes.

## Core Concepts
- Lucide icons are React components that render SVGs.
- You can scale icons by passing the `size` prop or via Tailwind classes.
- Icons behave like inline SVGs, so you can style them with `stroke`, `fill`, and CSS utilities.

## Quick Start
```tsx
import { LoaderIcon } from 'lucide-react'

export const Example = () => (
	<LoaderIcon className="animate-spin text-blue-600" size={24} />
)
```


## Examples

### Inline Button Spinner
Show a spinner while an async action runs.
```tsx
import { useState } from 'react'
import { LoaderIcon } from 'lucide-react'

export const SaveButton = () => {
	const [saving, setSaving] = useState(false)

	const handleSave = async () => {
		setSaving(true)
		await fakeRequest()
		setSaving(false)
	}

	return (
		<button
			onClick={handleSave}
			className="flex items-center gap-2 rounded bg-green-600 px-4 py-2 text-white disabled:opacity-50"
			disabled={saving}
		>
			{saving && <LoaderIcon size={16} className="animate-spin" />}
			<span>{saving ? 'Savingâ€¦' : 'Save'}</span>
		</button>
	)
}

const fakeRequest = () => new Promise(res => setTimeout(res, 1500))
```
The icon appears only while the promise resolves, giving clear feedback to the user.

### Centered Page Loader
```tsx
import { LoaderIcon } from 'lucide-react'

export const PageLoader = () => (
	<div className="flex h-screen items-center justify-center">
		<LoaderIcon className="animate-spin text-gray-500" size={48} />
	</div>
)
```
Use this pattern when fetching data for an entire page.

## Best Practices & Gotchas
- Combine `animate-spin` with Tailwind color utilities for instant theming.
- Keep spinner sizes consistent across the app to avoid visual noise.
- Hide spinners quickly once loading completes; lingering spinners confuse users.
- Use descriptive labels or aria attributes for accessibility when the spinner is the only visible element.
