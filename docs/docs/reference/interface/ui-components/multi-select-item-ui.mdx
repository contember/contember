# MultiSelectItemUI

## Overview
MultiSelectItemUI wraps one selected value inside a MultiSelect input.  
It groups the item content and related triggers (remove, drag-handle) so you can style or reorder the selection consistently.

## Styling Guidelines
* All snippets below use Tailwind CSS.  
* No mandatory Contember theme classes exist for this component. Adjust Tailwind classes to match your design system.

## Core Concepts
* Wrap each selected value with `<MultiSelectItemUI>`.
* Nest content in `<MultiSelectItemContentUI>` to display the label.
* Add `<MultiSelectItemRemoveButtonUI>` to let users unselect a value.
* Insert sortable or drag overlay variants for drag-and-drop reordering.

## Quick Start
```tsx
import {
  MultiSelectItemUI,
  MultiSelectItemContentUI,
  MultiSelectItemRemoveButtonUI,
} from '@contember/ui'

<MultiSelectItemUI className="flex items-center gap-1 rounded bg-gray-100 px-2 py-1">
  <MultiSelectItemContentUI className="text-sm">Company A</MultiSelectItemContentUI>
  <MultiSelectItemRemoveButtonUI
    onClick={e => e.stopPropagation()}
    className="text-gray-500 hover:text-red-500"
  />
</MultiSelectItemUI>
```

## Sub-Components / Building Blocks

### MultiSelectItemContentUI
Wraps the human-readable label of the selected entity.

### MultiSelectItemRemoveButtonUI
Clickable icon/button that removes the item from the selection.  
Call `stopPropagation()` in its `onClick` to avoid toggling the parent popover.

### MultiSelectItemDragOverlayUI
Overlay rendered while the item is dragged. Combine with `dnd-kit` or Contember repeater helpers.

### MultiSelectSortableItemContentUI
Drop-in replacement for `MultiSelectItemContentUI` when items are sortable.
