## Styling Guidelines
All examples use Tailwind CSS.  
The component follows Contember’s default theme; no additional theme classes are required.

## Core Concepts
- Multi-select inputs can render each selected item as a “chip”.
- Items become draggable when wrapped in a sortable context.
- `MultiSelectSortableItemContentUI` holds the visual content inside each draggable item.
- You pass any React nodes as `children`; the component adds no extra props.

## Quick Start
```tsx
import { MultiSelectSortableItemContentUI } from '@contember/ui'

function TagsField({ selectedTags }: { selectedTags: string[] }) {
	return (
		<ul className="flex flex-wrap gap-2">
			{selectedTags.map(tag => (
				<li key={tag}>
					<MultiSelectSortableItemContentUI className="px-3 py-1 bg-blue-600 text-white rounded">
						{tag}
					</MultiSelectSortableItemContentUI>
				</li>
			))}
		</ul>
	)
}
```

## Examples
### Sortable Chips With Remove Buttons
```tsx
import {
	MultiSelectItemUI,
	MultiSelectItemRemoveButtonUI,
	MultiSelectSortableItemContentUI,
} from '@contember/ui'

function SortableTags({ tags, onRemove }: { tags: string[]; onRemove: (tag: string) => void }) {
	return (
		<ul className="flex flex-wrap gap-2">
			{tags.map(tag => (
				<li key={tag}>
					<MultiSelectItemUI className="flex items-center gap-2 rounded bg-gray-200 px-2 py-1">
						<MultiSelectSortableItemContentUI>{tag}</MultiSelectSortableItemContentUI>
						<MultiSelectItemRemoveButtonUI
							className="text-gray-600 hover:text-red-600"
							onClick={() => onRemove(tag)}
						/>
					</MultiSelectItemUI>
				</li>
			))}
		</ul>
	)
}
```
The example renders each selected tag as a sortable chip.  
`MultiSelectSortableItemContentUI` wraps the tag text, while `MultiSelectItemRemoveButtonUI` removes a tag.

## Best Practices & Gotchas
- Always nest the component inside `MultiSelectItemUI` for proper styling.
- Pass only inline-safe content (text or small icons) as `children`.
- Avoid heavy interactions inside the content; it may hinder drag performance.
- Combine with `MultiSelectItemDragOverlayUI` to give users clear feedback during drag.
