# OrderedListRenderer

## Overview
OrderedListRenderer outputs an ordered list element coming from the Contember Slate editor.  
Use it in your `renderElement` callback to keep list rendering consistent across your app.

## Styling Guidelines
* All code examples use Tailwind CSS.  
* OrderedListRenderer adds no CSS classes of its own. Add any needed theme classes on a wrapping element.

## Core Concepts
* You receive the `element` prop from Slate during rendering.  
* OrderedListRenderer only renders; it does not handle editing logic.  
* Style through external classes instead of modifying the component.

## Quick Start
```tsx
import { OrderedListRenderer } from './elements/OrderedListRenderer'
import { RenderElementProps } from 'slate-react'

const renderElement = ({ element, children, attributes }: RenderElementProps) => {
	if (element.type === 'ordered-list') {
		return (
			<ol {...attributes} className="list-decimal pl-6">
				<OrderedListRenderer element={element} />
			</ol>
		)
	}
	return <p {...attributes}>{children}</p>
}
```

## Examples

### Custom counter style
Use Tailwind to switch the list marker style:

```tsx
<ol className="list-[upper-roman] pl-6">
	<OrderedListRenderer element={element} />
</ol>
```

The `list-[upper-roman]` utility shows Roman numerals instead of decimal numbers.

### Nested lists with spacing
```tsx
<ol className="list-decimal pl-6 space-y-1">
	<OrderedListRenderer element={element} />
</ol>
```

`space-y-1` keeps the vertical rhythm when list items contain block elements.

## Best Practices & Gotchas
* Pass the exact `element` object from Slate; do not clone or mutate it.  
* Wrap the renderer in an `<ol>` and apply list-related classes there.  
* Keep formatting concerns outside the renderer to simplify maintenance.
