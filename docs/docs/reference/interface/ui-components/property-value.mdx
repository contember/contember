## Core Concepts
* PropertyValue displays a single value inside a PropertyList.
* You combine PropertyValue with PropertyRow or PropertyItem for aligned layouts.
* Children define the rendered content; no extra props are needed.

## Quick Start
```tsx
import { PropertyList, PropertyRow, PropertyLabel, PropertyValue } from '~/lib/ui/property-list'

export const EmailPreview = () => (
	<PropertyList>
		<PropertyRow>
			<PropertyLabel className="font-semibold">Subject</PropertyLabel>
			<PropertyValue className="font-normal">Quarterly Report</PropertyValue>
		</PropertyRow>
	</PropertyList>
)
```

## Sub-Components / Building Blocks
### PropertyList
* Wraps a group of property rows or items.
* Provides vertical spacing.

### PropertyRow
* Aligns a label and value horizontally.

### PropertyLabel
* Renders the left-hand label.
* Usually bold or semibold.

### PropertyItem
* Shortcut that combines label and children in one tag.

Above you render an email record with labels and values neatly aligned.

## Best Practices & Gotchas
* Keep labels short; wrap long text inside PropertyValue, not PropertyLabel.
* Add `className="font-normal"` to PropertyValue to reset inherited bold styles.
* Group related rows in one PropertyList for consistent spacing.
