# ScrollTargetRenderer

## Overview
ScrollTargetRenderer renders a single `ScrollTargetElement` inside a Contember Slate editor.  
Use it when you need to output a scroll anchor that other components (e.g. a table of contents) can reference.

## Styling Guidelines
* All code samples use Tailwind CSS.  
* ScrollTargetRenderer adds no visual styles by default, so wrap it in your own Tailwind classes if needed.

## Core Concepts
* Slate editors render their content through element renderers.  
* ScrollTargetRenderer expects a `ScrollTargetElement` instance from Slate.  
* The component itself produces an invisible anchor; styling and positioning are up to you.  

## Quick Start
```tsx
import { ScrollTargetRenderer } from '@contember/react-slate-editor'

// `element` is supplied by Slate's renderElement callback
<ScrollTargetRenderer element={element} />
```
