# SelectContent

## Overview
_None_

## Styling Guidelines
* All snippets in this page use Tailwind utility classes.  
* Contember’s default theme defines helpers such as `text-muted-foreground`; some examples rely on them.

## Core Concepts
* SelectContent renders the dropdown panel of a `Select`.
* It wraps one or more `SelectItem` or `SelectGroup` elements.
* You position the panel with the `align` prop.
* Tailwind classes added through `className` style the panel.

## Quick Start
```tsx
import { Select, SelectContent, SelectItem, SelectTrigger } from '~/lib/ui/select'

export const Example = () => (
	<Select>
		<SelectTrigger className="w-40 border rounded px-2 py-1">Choose</SelectTrigger>

		<SelectContent className="border rounded-md bg-white shadow-md">
			<SelectItem value="apple">Apple</SelectItem>
			<SelectItem value="banana">Banana</SelectItem>
		</SelectContent>
	</Select>
)
```

## Examples

```tsx
// Country picker with flag, dial code and name
<Select>
	<SelectTrigger className="w-48 border rounded px-2 py-1">Choose country</SelectTrigger>

	<SelectContent className="w-72">
		<SelectGroup>
			{COUNTRIES.map(country => (
				<SelectItem key={country.code} value={country.code}>
					<span className="flex items-center gap-2">
						<span className="text-base">{country.flag}</span>
						<span>{country.dialCode}</span>
						<span className="text-muted-foreground text-xs">{country.name}</span>
					</span>
				</SelectItem>
			))}
		</SelectGroup>
		<SelectGroup>
			<SelectItem value="add-custom">
				<span className="flex items-center gap-2">
					<PlusCircle className="w-4 h-4" />
					<span>Other country</span>
				</span>
			</SelectItem>
		</SelectGroup>
	</SelectContent>
</Select>
```
The example shows how you can combine `SelectGroup` and complex item layouts inside `SelectContent`.

```tsx
// Locale switcher aligned to the start of the trigger
<Select value={currentLocale} onValueChange={handleLocaleChange}>
	<SelectTrigger className="w-18">
		<LocalePreview localeCode={currentLocale} showCode={false} />
	</SelectTrigger>

	<SelectContent align="start" className="min-w-0">
		{locales?.map(locale => (
			<SelectItem key={locale.code} value={locale.code}>
				<LocalePreview localeCode={locale.code} showCode={false} />
			</SelectItem>
		))}
	</SelectContent>
</Select>
```
Here, `align="start"` displays the dropdown flush with the left edge of the trigger.

## Best Practices & Gotchas
* Keep `SelectContent` as a direct child of `Select` to ensure focus management works.
* Use Tailwind spacing utilities (`gap-*`, `p-*`) inside items instead of nested `<div>` elements.
* Always set an explicit width (`w-*`) or min-width on `SelectContent` if items contain long text.
* Pass `align="start"` when the trigger is narrow to avoid content overflow.
