# SelectScrollDownButton

## Overview
SelectScrollDownButton renders a clickable control that scrolls selectable content downward.  
Use it inside Contember Select components to reveal items that overflow the visible area.

## Styling Guidelines
All examples use Tailwind CSS.  
No additional Contember theme classes are required.

## Core Concepts
- The button appears below a scrollable select list.  
- Clicking the button advances the list by a fixed offset.  
- It has no public props; behavior is context-driven.

## Quick Start
```tsx
import { SelectScrollDownButton } from "@contember/react-select";

export const BasicSelect = () => (
	<div className="w-64">
		{/* ...your SelectInput here... */}
		<SelectScrollDownButton className="w-full py-2 text-sm text-gray-600 hover:bg-gray-100" />
	</div>
);
```
