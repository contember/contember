## Core Concepts
* `SelectTrigger` toggles the visibility of `SelectContent`.
* You wrap `SelectTrigger` inside a `Select` component.
* Combine `SelectValue` and Tailwind classes for layout and sizing.

## Quick Start
```tsx
import { Select, SelectTrigger, SelectContent, SelectItem, SelectValue } from '~/lib/ui/select'

export const QuickStart = () => (
	<Select onValueChange={value => console.log(value)}>
		<SelectTrigger className="w-48">
			<SelectValue placeholder="Choose an option" />
		</SelectTrigger>

		<SelectContent>
			<SelectItem value="a">Option A</SelectItem>
			<SelectItem value="b">Option B</SelectItem>
		</SelectContent>
	</Select>
)
```

## Detailed Guide
1. Wrap `SelectTrigger` inside `Select`.  
2. Add `className` to control width, borders, or spacing.  
3. Place any visual preview inside `SelectValue`.  
4. Use `SelectItem` to list options.  

```tsx
<Select>
	<SelectTrigger className="flex items-center gap-2 px-3 py-2 border rounded-md">
		<SelectValue placeholder="Pick one" />
	</SelectTrigger>
	<SelectContent>
		<SelectItem value="red">Red</SelectItem>
		<SelectItem value="green">Green</SelectItem>
		<SelectItem value="blue">Blue</SelectItem>
	</SelectContent>
</Select>
```

## Examples

### Locale switcher
```tsx
const { locales, currentLocale } = useLocale()

<Select value={currentLocale} onValueChange={setLocale}>
	<SelectTrigger className="w-20">
		<LocalePreview localeCode={currentLocale} showCode={false} />
	</SelectTrigger>
	<SelectContent align="start" className="min-w-0">
		{locales.map(locale => (
			<SelectItem key={locale.code} value={locale.code}>
				<LocalePreview localeCode={locale.code} showCode={false} />
			</SelectItem>
		))}
	</SelectContent>
</Select>
```
The trigger shows a flag icon. Selecting a new locale reloads the page with the chosen language.

## Best Practices & Gotchas
* Add `className` directly on `SelectTrigger`; styling `Select` does nothing.  
* Keep the trigger width fixed to avoid layout jumps when the value text changes.  
* Place only small JSX inside `SelectValue`; large content can break the dropdown layout.  
* When disabling the select, use the `Select` prop, not the trigger.
