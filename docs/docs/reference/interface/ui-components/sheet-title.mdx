## Styling Guidelines  
All examples use Tailwind CSS. You do not need any special Contember theme classes for `SheetTitle`.

## Core Concepts  
- `SheetTitle` renders a heading inside a `Sheet`.  
- It helps you keep typography consistent across sheets.  
- You can extend its look with Tailwind utility classes.  
- Place it near the top of your sheet for optimal accessibility.

## Quick Start  

```tsx
import { Sheet, SheetTitle } from '@contember/ui'

export const Demo = () => (
	<Sheet isOpen>
		<SheetTitle className="text-xl font-semibold">
			Invoice details
		</SheetTitle>
		{/* ...sheet content... */}
	</Sheet>
)
```

## Best Practices & Gotchas  
- Keep the title short and descriptive.

- Place only text inside `SheetTitle`; avoid complex markup.  
- Do not duplicate titles inside the same sheet.
