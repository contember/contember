## Styling Guidelines
* All examples in this page use Tailwind CSS utility classes.
* The component sits inside a `<Sidebar>` that applies Contember theme classes such as `bg-sidebar`, `text-sidebar`, and `text-sidebar-accent`.

## Core Concepts
* SidebarContent wraps the scrollable middle section of a Contember sidebar.
* It holds one or more `<SidebarGroup>` elements that organize navigation or actions.
* You usually pair it with `<SidebarHeader>` and `<SidebarFooter>` for a complete sidebar layout.
* The component accepts standard `div` props (className, style, …) through React’s prop spreading.

## Quick Start
```tsx
import {
  Sidebar,
  SidebarContent,
  SidebarGroup,
} from '~/lib/ui/sidebar'

export default function BasicSidebar() {
  return (
    <Sidebar className="h-screen w-64 bg-sidebar text-sidebar">
      <SidebarContent className="px-4 py-6 space-y-4 overflow-y-auto">
        <SidebarGroup>
          <a className="block py-2 text-sm hover:text-sidebar-accent">Dashboard</a>
        </SidebarGroup>
        <SidebarGroup>
          <a className="block py-2 text-sm hover:text-sidebar-accent">Settings</a>
        </SidebarGroup>
      </SidebarContent>
    </Sidebar>
  )
}
```

## Examples

### Scrollable navigation with header and footer
```tsx
import {
  Sidebar,
  SidebarHeader,
  SidebarContent,
  SidebarGroup,
  SidebarGroupContent,
  SidebarFooter,
} from '~/lib/ui/sidebar'

export function AppSidebar() {
  return (
    <Sidebar className="h-screen w-64 bg-sidebar text-sidebar">
      <SidebarHeader className="p-4">
        <h1 className="text-lg font-semibold">Contember</h1>
      </SidebarHeader>

      <SidebarContent>
        <SidebarGroup>
          <SidebarGroupContent>
            <nav className="space-y-1 px-4">
              <a className="block py-2 hover:text-sidebar-accent">Home</a>
              <a className="block py-2 hover:text-sidebar-accent">Projects</a>
            </nav>
          </SidebarGroupContent>
        </SidebarGroup>
      </SidebarContent>

      <SidebarFooter className="p-4">
        <small className="block text-xs">&copy; 2024 Contember</small>
      </SidebarFooter>
    </Sidebar>
  )
}
```
The header stays fixed, SidebarContent scrolls, and the footer stays fixed at the bottom.

### Combining navigation and quick actions
```tsx
import {
  Sidebar,
  SidebarContent,
  SidebarGroup,
  SidebarMenu,
  SidebarMenuItem,
  SidebarMenuButton,
} from '~/lib/ui/sidebar'
import { LifeBuoyIcon, CogIcon } from 'lucide-react'

export function SupportSidebar() {
  return (
    <Sidebar className="h-screen w-72 bg-sidebar text-sidebar">
      <SidebarContent className="py-8 space-y-6">
        <SidebarGroup>
          <SidebarMenu>
            <SidebarMenuItem>
              <SidebarMenuButton asChild>
                <a href="#" className="flex items-center gap-2 px-4 py-2">
                  <LifeBuoyIcon className="w-4 h-4" />
                  Support
                </a>
              </SidebarMenuButton>
            </SidebarMenuItem>
            <SidebarMenuItem>
              <SidebarMenuButton asChild>
                <a href="#" className="flex items-center gap-2 px-4 py-2">
                  <CogIcon className="w-4 h-4" />
                  Settings
                </a>
              </SidebarMenuButton>
            </SidebarMenuItem>
          </SidebarMenu>
        </SidebarGroup>
      </SidebarContent>
    </Sidebar>
  )
}
```
SidebarContent holds both navigation items and quick-action buttons, showing its flexibility.

## Best Practices & Gotchas
* Always place SidebarContent between SidebarHeader and SidebarFooter for predictable layout.
* Apply `overflow-y-auto` on SidebarContent when the sidebar can grow taller than the viewport.
* Keep padding consistent (`px-4 py-6`) so content aligns with header and footer.
* Do not nest multiple SidebarContent components inside one Sidebar; use SidebarGroup for sectioning.
