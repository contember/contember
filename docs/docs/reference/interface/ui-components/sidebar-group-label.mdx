# SidebarGroupLabel

## Overview
`SidebarGroupLabel` renders a heading for a logical group inside a Contember sidebar navigation.
You wrap any `ReactNode` inside it—most commonly plain text.

## Styling Guidelines
* All examples use Tailwind CSS.
* The component relies on Contember’s default sidebar theme; no extra classes are required.

## Core Concepts
* Sidebar navigation often needs visual grouping.
* A label improves readability and keyboard navigation.
* The component is presentational and accepts only children.

## Quick Start
```tsx
import { SidebarGroupLabel } from '~/lib/ui/sidebar'

export const Navigation = () => (
	<aside className="w-64 bg-gray-900 text-white">
		<SidebarGroupLabel>Content</SidebarGroupLabel>
		{/* SidebarMenu goes here */}
	</aside>
)
```

## Examples

### Render a collapsible menu with a label
```tsx
import {
	SidebarGroupLabel,
	SidebarMenu,
	SidebarItem,
} from '~/lib/ui/sidebar'

export const CollapsibleMenu = () => (
	<>
		<SidebarGroupLabel>Administration</SidebarGroupLabel>
		<SidebarMenu>
			<SidebarItem href="/users">Users</SidebarItem>
			<SidebarItem href="/roles">Roles</SidebarItem>
		</SidebarMenu>
	</>
)
```
The label separates administration links from the rest of the navigation.
Users can quickly scan the menu for relevant sections.

### Using stored state to remember opened groups
```tsx
const Navigation = ({ children, label }: { children?: React.ReactNode; label?: React.ReactNode }) => {
	const [menuCollapsibleState, setMenuCollapsibleState] = useStoredState<{ openedItems: string[]; closedItems: string[] }>(
		'session',
		['navigation', 'navState'],
		value => ({ openedItems: [], closedItems: [], ...value }),
	)

	return (
		<MenuContext.Provider value={{ level: 0 }}>
			<OpenedItemsContext.Provider value={{ menuCollapsibleState, setMenuCollapsibleState }}>
				{label && <SidebarGroupLabel>{label}</SidebarGroupLabel>}
				<SidebarMenu>{children}</SidebarMenu>
			</OpenedItemsContext.Provider>
		</MenuContext.Provider>
	)
}
```
This pattern keeps the collapsible state in `sessionStorage`, ensuring the menu stays consistent across page loads.

## Best Practices & Gotchas
* Keep labels short—ideally one or two words.
* Do not nest interactive elements inside the label.
* Place the label directly above the related `SidebarMenu`.
