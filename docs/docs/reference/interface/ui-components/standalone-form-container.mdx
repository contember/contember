# StandaloneFormContainer

## Overview
StandaloneFormContainer wraps one or more form fields and renders an optional label.  
The component is **deprecated**; switch to `FormContainer` to receive updates and bug fixes.

## Styling Guidelines
* All examples use Tailwind utility classes.  
* No Contember–specific theme classes are required.

## Core Concepts
* Container groups a label and its children in a consistent layout.  
* Accepts any React nodes as children, not only Contember fields.  
* Supports an optional `labelPosition` to align the label.

## Quick Start
```tsx
import { InputField, StandaloneFormContainer } from '@app/lib/form'

export const NameField = () => (
  <StandaloneFormContainer label="Name">
    <InputField field="name" className="w-full" />
  </StandaloneFormContainer>
)
```

## Examples

### Display a read-only value with formatting
```tsx
<StandaloneFormContainer label="Status">
  <Field<string> field="status" format={v => v ?? '—'} />
</StandaloneFormContainer>
```
The container shows the formatted value while preserving consistent label spacing.

### Toggle a date value with a checkbox
```tsx
const UpdatedAtToggle = ({ label, field }: { label: string; field: string }) => {
  const f = useField<string | null>(field)
  return (
    <StandaloneFormContainer
      label={
        <>
          {label}
          {f.value && (
            <i className="text-muted-foreground font-normal">
              ({new Date(f.value).toLocaleDateString()})
            </i>
          )}
        </>
      }
      labelPosition="right"
    >
      <CheckboxInput
        checked={!!f.value}
        onChange={e => f.updateValue(e.target.checked ? new Date().toISOString() : null)}
      />
    </StandaloneFormContainer>
  )
}
```
The label dynamically displays the current date, and the checkbox updates the ISO timestamp.

## Best Practices & Gotchas
* Migrate to `FormContainer`; future releases will drop this component.  
* Keep labels concise to maintain alignment when `labelPosition="left"`.  
* Wrap complex child layouts in a `div` with `flex flex-col gap-1` to avoid cramped spacing.
