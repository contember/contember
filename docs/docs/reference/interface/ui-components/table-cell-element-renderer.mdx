# TableCellElementRenderer

## Overview
`TableCellElementRenderer` renders a single table cell inside the Contember Slate editor.  
You pass a `TableCellElement` object to control the cell’s content and attributes.

## Styling Guidelines
* All examples use Tailwind CSS classes.
* No additional Contember theme classes are required.

## Core Concepts
* Slate elements represent editor nodes as plain objects.
* `TableCellElement` describes cell content, `colspan`, and `rowspan`.
* `TableCellElementRenderer` converts that object to React JSX.

## Quick Start
```tsx
import { TableCellElementRenderer } from '@contember/react-slate'
import { TableCellElement } from '@contember/react-slate'

const cell: TableCellElement = {
  type: 'tableCell',
  children: [{ text: 'Hello' }],
}

export const Demo = () => (
  <TableCellElementRenderer
    element={cell}
    className="border p-2"
  />
)
```

## Examples

### Editable Table Cell
```tsx
import { Slate, withReact } from 'slate-react'
import { createEditor } from 'slate'
import { TableCellElementRenderer } from '@contember/react-slate'

const editor = withReact(createEditor())

const initialCell: TableCellElement = {
  type: 'tableCell',
  children: [{ text: 'Editable content' }],
}

export const EditableCell = () => (
  <Slate editor={editor} value={[initialCell]}>
    <TableCellElementRenderer
      element={initialCell}
      className="border border-gray-300 p-3 focus:outline-none"
    />
  </Slate>
)
```
This example shows how to embed the renderer inside a Slate context so users can edit the cell.

## Best Practices & Gotchas
* Always wrap the renderer in a `Slate` provider when you need editing.
* Keep `element` immutable; update through Slate transforms.
* Add Tailwind border classes for visible table grid lines.
