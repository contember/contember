## Styling Guidelines

## Core Concepts
* You pass a single `TableElement` object to render the table.  
* You place the component inside a Slate editor context.

## Quick Start
```tsx
import { TableElementRenderer } from './elements/TableElementRenderer'
import type { TableElement } from '@contember/react-slate-editor'

const element: TableElement = /* obtain from your Slate editor state */

export const Example = () => (
  <div className="overflow-auto">
    <TableElementRenderer element={element} />
  </div>
)
```
