## Overview
TableRow renders a single HTML `<tr>` element inside Contember’s `Table` component.  
Use it to group `TableHead` or `TableCell` elements into a logical row.

## Styling Guidelines
All examples use Tailwind CSS classes.  
TableRow itself needs no special Contember theme classes, but it inherits border and spacing utilities applied to the parent `Table`.

## Core Concepts
- TableRow represents one row within a Contember table.
- Combine TableRow with `TableHead` or `TableCell` to build headers and data rows.
- You can pass any valid HTML `<tr>` attributes (e.g., `className`, `onClick`).

## Quick Start
```tsx
import { Table, TableHeader, TableRow, TableHead } from '~/lib/ui/table'

export const BasicTable = () => (
  <Table className="w-full border-collapse">
    <TableHeader>
      <TableRow className="bg-gray-100">
        <TableHead className="p-2 text-left">Name</TableHead>
        <TableHead className="p-2 text-left">Email</TableHead>
      </TableRow>
    </TableHeader>
  </Table>
)
```

## Examples

### Header Row with Entity Labels
```tsx
<TableRow>
  <TableHead>{entityLabels.HealthRiskPeriod.fields.ageFrom}</TableHead>
  <TableHead>{entityLabels.HealthRiskPeriod.fields.ageTo}</TableHead>
  <TableHead>{entityLabels.HealthRiskPeriod.fields.intervalMonths}</TableHead>
</TableRow>
```
The row displays translated header labels fetched from an `entityLabels` helper.

### Data Row with Formatted Fields
```tsx
<TableRow>
  <TableCell>
    <FormattedField field="ageFrom" />
  </TableCell>
  <TableCell>
    <FormattedField field="ageTo" />
  </TableCell>
  <TableCell>
    <FormattedField field="intervalMonths" />
  </TableCell>
</TableRow>
```
Each cell renders a formatted value using Contember’s `FormattedField` component.

### Row Without Bottom Border
```tsx
<TableRow className="last:border-b-0">
  <TableCell>
    <Field field="details.firstName" /> <Field field="details.lastName" />
  </TableCell>
  <TableCell>
    <Field field="details.email" />
  </TableCell>
  <TableCell>
    <Field field="details.phone" />
  </TableCell>
</TableRow>
```
The `last:border-b-0` utility removes the bottom border from the final row.

## Best Practices & Gotchas
- Keep cell counts consistent across header and body rows.
- Use Tailwind utilities like `last:border-b-0` to fine-tune borders.
- Pass `onClick` to TableRow for row-level interactions such as navigation.
- Avoid placing interactive elements directly on TableRow; use cells instead.
