# TextareaAutosize

## Overview
_None_

## Styling Guidelines
* All examples use Tailwind utility classes.  
* The component does not require any Contember-specific theme classes, but it works well with common classes such as `border`, `rounded`, and `w-full`.

## Core Concepts
* The textarea grows vertically to fit its content.
* You can control the value with React state or let the browser manage it.
* The component forwards all native `<textarea>` attributes and events.

## Quick Start
```tsx
import { useState } from 'react'
import { TextareaAutosize } from '@app/lib/ui/textarea'

export const BasicDemo = () => {
  const [note, setNote] = useState('')

  return (
    <TextareaAutosize
      value={note}
      onChange={e => setNote(e.target.value)}
      placeholder="Type your notes…"
      className="w-full border rounded p-2"
    />
  )
}
```

The component integrates seamlessly with Contember forms. It receives `value` and `onChange` like a regular textarea, while Tailwind classes handle the visual styling.

## Best Practices & Gotchas
* Keep the component controlled to avoid cursor jumps when syncing external state.
* Limit `maxRows` (if supported) to prevent the textarea from overtaking the page.
* Combine with `resize-none` Tailwind class if you want to disable manual resizing.
