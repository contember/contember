# UploaderRepeaterDragOverlayUI

## Overview
`UploaderRepeaterDragOverlayUI` renders the visual overlay while you drag an uploader item inside a sortable repeater list.  
It works with `RepeaterSortableDragOverlay` to show a floating preview that follows the cursor.

## Styling Guidelines
* All examples use Tailwind CSS utility classes.  
* No special Contember theme classes are required; extend with your own classes if needed.

## Core Concepts
* Overlay appears only during drag operations.
* Renders any children you pass, so structure it exactly like a normal list item.
* Must sit inside `RepeaterSortableDragOverlay` to receive drag-layer styles.

## Quick Start
```tsx
import { UploaderRepeaterDragOverlayUI } from '@contember/react-components'

export const FilesField = () => (
	<RepeaterSortableDragOverlay>
		<UploaderRepeaterDragOverlayUI className="rounded-md bg-white shadow-lg p-4">
			{/* Your uploader preview goes here */}
		</UploaderRepeaterDragOverlayUI>
	</RepeaterSortableDragOverlay>
)
```

## Detailed Guide
```tsx
import {
	RepeaterSortable,
	RepeaterSortableDragOverlay,
	UploaderRepeaterItemUI,
	UploaderRepeaterDragOverlayUI,
} from '@contember/react-components'

export const SortableUploads = () => (
	<RepeaterSortable>
		{/* ...items rendered here... */}

		<RepeaterSortableDragOverlay>
			<UploaderRepeaterDragOverlayUI className="pointer-events-none">
				<UploaderRepeaterItemUI className="flex items-center gap-2">
					{/* Thumbnail + filename */}
				</UploaderRepeaterItemUI>
			</UploaderRepeaterDragOverlayUI>
		</RepeaterSortableDragOverlay>
	</RepeaterSortable>
)
```

## Examples

### Drag overlay with thumbnail
```tsx
<RepeaterSortableDragOverlay>

		<img src={thumbnailUrl} className="h-12 w-12 object-cover rounded" />
		<span className="truncate max-w-[200px]">{fileName}</span>
	</UploaderRepeaterDragOverlayUI>
</RepeaterSortableDragOverlay>
```
The overlay shows a thumbnail and filename while you drag an item.

### Minimal overlay
```tsx
<UploaderRepeaterDragOverlayUI>
	{children /* identical markup to normal list item */}
</UploaderRepeaterDragOverlayUI>
```
Pass exactly the same children as the normal item so the overlay mirrors it.

## Best Practices & Gotchas
* Keep overlay markup identical to the list item for a seamless illusion.
* Add `pointer-events-none` so the overlay never intercepts mouse events.
* Use rounded corners and shadow to hint that the element is floating.
* Do not place stateful hooks inside the overlay; it re-mounts frequently.
